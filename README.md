# dotfiles

tmux linux configuration files

## tmux keycuts

    <c-a> &                  kill window
    <c-a> x                  kill panel
    <c-a> number             switch windows
    <c-a> p                  previous window
    <c-a> n                  next window
    <c-a> %                  split windows h
    <c-a> "                  split windows v
    <c-a> c                  new window
